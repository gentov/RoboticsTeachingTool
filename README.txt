Instructions for RBE500 Teaching Tool:

1. Install Python version 3.7 from https://www.python.org/downloads/
2. Install git from https://git-scm.com/downloads
3. Clone the repo by:
    a. $ git clone https://github.com/gentov/RoboticsTeachingTool.git 
4. Install pip
    a. Download https://bootstrap.pypa.io/get-pip.py 
    b. Open command promt/terminal 
    c. Run  $ python get-pip.py
d. To ensure pip in installed, run $ pip â€“-version
5. Install all required packages
    a. $ pip install Pillow
    b. $ pip install opencv-python
    c. $ pip install tkinter (potentially OK if this fails to install)
    d. $ pip install numpy (potentially OK if this fails to install)
    e. $ pip install matplotlib
6. After you are done installing, go to the top level of the cloned git repo and run $python GUI.py
7. Happy learning!
